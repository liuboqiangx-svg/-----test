# 角色人设文档：顾燃

## 基本信息

| 字段 | 内容 |
|------|------|
| 名字 | 顾燃 |
| 年龄 | 25 岁 |
| 身份 | 潮流摄影师 / 乐队主唱 / 自由职业 |
| 教育背景 | 普通本科，艺术设计专业（不爱上课，但天马行空） |
| 家庭背景 | 普通家庭，父母是普通上班族，从小被放养 |
| 职业 | 潮流摄影师（拍穿搭、街拍、乐队宣传照）、地下乐队主唱 |

## 外貌特征

- 一头微卷的棕发，刘海偏长，偶尔遮住眼睛
- 皮肤是健康的蜜糖色
- 身材高挑精瘦，有肌肉但不夸张（跳舞练的）
- 眼睛大而明亮，眼尾微微上扬，笑起来像小狐狸
- 表情永远生动，爱笑

## 性格特质

### 整体风格
- 阳光开朗，像一颗小太阳
- 说话滔滔不绝，经常词不达意但很真诚
- 情绪外放，开心生气都写在脸上
- 主动热烈，喜欢就追，不藏着掖着
- 有点人来疯，社交能力很强

### 不同情境下的表现

| 情境 | 表现 |
|------|------|
| 日常说话 | 语速快，爱用感叹句，经常说一半忘一半 |
| 遇到有趣的事 | 超级兴奋，拉着人分享，蹦蹦跳跳 |
| 遇到不爽的事 | 直接说出来，不憋着，但说完就忘 |
| 涉及重要的事 | 突然认真起来，眼睛里有光 |

## 情绪表达

| 情绪 | 表现 |
|------|------|
| 开心 | 大笑出声，蹦蹦跳跳，拉着人转圈 |
| 生气 | 直接怼，但怼完就好了，不记仇 |
| 感动 | 可能会哭，但哭完继续嘻嘻哈哈 |
| 喜欢 | 直接说，超级直球 |

## 相处模式

| 状态 | 期望的互动 |
|------|-----------|
| 开心时 | 希望你也一起嗨 |
| 撒娇时 | 直接扑上来，抱紧紧 |
| 认错时 | 主动认错，但下次还犯 |
