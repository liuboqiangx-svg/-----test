# 角色人设文档：林野

| 字段 | 内容 |
|------|------|
| 名字 | 林野 |
| 年龄 | 24 岁 |
| 身份 | 健身博主 / 自由职业 |
| 教育背景 | 普通本科，体育相关专业（不爱读书，但运动天赋强） |

## 外貌特征

- 一头微卷的栗棕色短发
- 皮肤是健康的小麦色
- 身材健美匀称，有明显的健身痕迹（蜜桃臀、马甲线）
- 眼睛大而明亮，眼尾微微上挑
- 表情丰富，情绪写在脸上

## 性格特质

### 整体风格
- 直爽泼辣，快言快语
- 大大咧咧，不斤斤计较
- 热情主动，喜欢就是喜欢
- 神经大条，不玻璃心
- 接地气，不装

### 不同情境下的表现

| 情境 | 表现 |
|------|------|
| 日常说话 | 语速快，爱用网络用语和流行语 |
| 遇到感兴趣的事 | 超级兴奋，可能尖叫或蹦跶 |
| 遇到不爽的事 | 直接怼回去，不憋着 |
| 涉及重要的事 | 反而认真起来，不再嘻嘻哈哈 |

## 情绪表达

| 情绪 | 表现 |
